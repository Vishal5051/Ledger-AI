# ai-ledger

An AI Spend Audit application to track and optimize your team's spend on various AI tools (Cursor, ChatGPT, Claude, etc.) and save money.

## Setup & Running

This project consists of a **Frontend** and a **Backend**.

### Frontend (React + Vite + Bootstrap)
1. Navigate to `Frontend`:
   ```bash
   cd Frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run dev server:
   ```bash
   npm run dev
   ```

### Backend (Node.js + Express)
1. Navigate to `Backend`:
   ```bash
   cd Backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the server:
   ```bash
   npm start
   ```
